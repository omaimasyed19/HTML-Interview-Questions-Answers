HTML Interview Questions & Answers

1. What is HTML?
HTML (HyperText Markup Language) is the standard language for creating web pages and web applications. It describes the structure of a web page using a system of tags and attributes.

 
                <!DOCTYPE html>
                <html>
                <head>
                    <title>HTML Example</title>
                </head>
                <body>
                    <h1>Hello, World!</h1>
                    <p>This is an example of HTML.</p>
                </body>
                </html>

2. What is XHTML?
XHTML (Extensible HyperText Markup Language) is a stricter, XML-based version of HTML. It follows stricter rules for tag closing and nesting.

4. What is DOM?
DOM (Document Object Model) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

                <!DOCTYPE html>
                <html>
                <head>
                    <title>DOM Example</title>
                </head>
                <body>
                    <p id="demo">Hello, DOM!</p>
                    <script>
                        document.getElementById("demo").innerHTML = "Hello, World!";
                    </script>
                </body>
                </html>

5. What does <!DOCTYPE html> do?
The `<!DOCTYPE html>` declaration defines the document type and version of HTML. It helps the browser render the page correctly.

7. Block-Level and Inline Elements
Block-level elements: <div>, <ol>, <ul>, <main>
Inline elements: <span>, <a>, <img>, <small>

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Block and Inline Elements</title>
                    <style>
                        .block { display: block; }
                        .inline { display: inline; }
                    </style>
                </head>
                <body>
                    <div class="block">This is a block element.</div>
                    <span class="inline">This is an inline element.</span>
                </body>
                </html>

8. Semantic and Non-Semantic Elements
Semantic elements: <article>, <section>, <nav>
Non-semantic elements: <div>, <span>

                <!DOCTYPE html>

                <head>

                </head>

                            <li><a href="#">Home</a></li>

                            <h2>Article Title</h2>
                            <p>This is an example of an article element in HTML5.</p>
                        </article>
                    </main>
                </body>
                </html>

9. Optional Tags and Self-Closing Tags
Optional tags: Some tags can be omitted (e.g., </li>, </p>).
Self-closing tags: Tags that do not need a closing tag (e.g., <img />, <input />).

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Optional and Self-Closing Tags</title>
                </head>
                <body>

                </body>
                </html>

10. Responsive Design
Responsive design ensures that web pages look good on all devices.

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Responsive Design</title>
                    <meta name="viewport" content="width=device-width, initial-scale=1">
                    <style>
                        body { font-family: Arial, sans-serif; }
                        .container { width: 100%; max-width: 1200px; margin: auto; }
                        .responsive-img { width: 100%; height: auto; }
                    </style>
                </head>
                <body>
                    <div class="container">
                        <h1>Responsive Design Example</h1>
                        <img src="image.jpg" alt="Example Image" class="responsive-img">
                    </div>
                </body>
                </html>

11. Character Encoding
Character encoding tells computers how to interpret digital data into text.

                <!DOCTYPE html>
                <html>
                <head>
                    <meta charset="UTF-8">
                    <title>Character Encoding</title>
                </head>
                <body>
                    <p>This document uses UTF-8 encoding.</p>
                </body>
                </html>

12. Meta Tags
Meta tags provide metadata about the HTML document.

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Meta Tags Example</title>
                    <meta name="description" content="An example of meta tags in HTML.">
                    <meta name="keywords" content="HTML, meta tags, example">
                </head>
                <body>
                    <h1>Meta Tags Example</h1>
                    <p>Check the source code to see the meta tags.</p>
                </body>
                </html>

13. alt Attribute for Images
The `alt` attribute provides alternative text for an image if it cannot be displayed.

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Alt Attribute Example</title>
                </head>
                <body>
                    <img src="image.jpg" alt="Example Image">
                </body>
                </html>

14. select vs. datalist
The `select` element presents a drop-down list of options. The `datalist` element provides a list of suggestions for an input field.

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Select vs Datalist</title>
                </head>
                <body>
                    <label for="browser">Choose your browser:</label>
                    <input list="browsers" id="browser" name="browser">
                    <datalist id="browsers">
                        <option value="Chrome">
                        <option value="Firefox">
                        <option value="Safari">
                        <option value="Edge">
                    </datalist>
                </body>
                </html>

15. Desktop First and Mobile First Design
Desktop First: Design for larger screens first, then adapt to smaller screens.
Mobile First: Design for smaller screens first, then adapt to larger screens.

16. Data Attributes
Data attributes store extra information on HTML elements.

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Data Attributes</title>
                </head>
                <body>
                    <div data-user-id="12345">User Info</div>
                    <script>
                        var userId = document.querySelector('div').getAttribute('data-user-id');
                        console.log(userId);
                    </script>
                </body>
                </html>

17. HTML Video Example
The `<video>` tag is used to embed video content.

                <!DOCTYPE html>
                <html>
                <head>
                    <title>HTML Video Example</title>
                </head>
                <body>
                    <video width="320" height="240" controls>

                    </video>
                </body>
                </html>

18. What is the <picture> element?
The `<picture>` element allows you to define multiple sources for an image. It is used to provide different images based on device or display conditions.

                    <picture>
                        <source srcset="image-large.jpg" media="(min-width: 800px)">
                        <source srcset="image-medium.jpg" media="(min-width: 500px)">
                        <img src="image-small.jpg" alt="Example Image">
                    </picture>

19. What is the <progress> element?
The `<progress>` element represents the completion progress of a task.

                    <progress value="70" max="100">70%</progress>

20. What is the <meter> element?
The `<meter>` element represents a scalar measurement within a known range, such as disk usage or the relevance of a query result.

                    <meter value="2" min="0" max="10">2 out of 10</meter>

21. What is the <details> element?
The `<details>` element is used to create a disclosure widget that users can open and close to reveal more information.

                    <details>
                        <summary>More Information</summary>
                        <p>This is additional details that can be revealed by clicking on the summary.</p>
                    </details>

22. What is the <dialog> element?
The `<dialog>` element represents a dialog box or other interactive component, such as an inspector or window.

                    <dialog open>
                        <p>This is a dialog box.</p>
                        <button>Close</button>
                    </dialog>

23. What is the <template> element?
The `<template>` element is a mechanism for holding client-side content that you donâ€™t want to be rendered when the page loads. It can be used for dynamic content using JavaScript.

                    <template id="my-template">
                        <div>This is a template content.</div>
                    </template>

24. What is the <slot> element?
The `<slot>` element is part of Web Components and is used to define placeholders inside a web component that can be filled with your own markup.

                    <my-component>
                        <span slot="content">This will go into the content slot.</span>
                    </my-component>

25. What is the <output> element?
The `<output>` element represents the result of a calculation or user action.

                    <form oninput="result.value=parseInt(a.value)+parseInt(b.value)">
                        <input type="range" id="a" value="50">
                        + <input type="number" id="b" value="50">
                        = <output name="result" for="a b">100</output>
                    </form>

26. What is the <time> element?
The `<time>` element represents a specific period in time.

                    <time datetime="2024-07-11T12:00">July 11, 2024, 12:00 PM</time>

27. What is the <mark> element?
The `<mark>` element represents highlighted text.

                    <p>This is a <mark>highlighted text</mark> example.</p>

28. What is the <abbr> element?
The `<abbr>` element represents an abbreviation or acronym, with an optional `title` attribute to provide the full form.

                    <abbr title="HyperText Markup Language">HTML</abbr>

29. What is the <figure> and <figcaption> elements?
The `<figure>` element represents self-contained content, like illustrations, diagrams, photos, code listings, etc., usually with a `<figcaption>` element to provide a caption.

                    <figure>
                        <img src="image.jpg" alt="Example Image">
                        <figcaption>This is an example image.</figcaption>
                    </figure>

30. What is the <cite> element?
The `<cite>` element represents the title of a work (e.g., book, report, thesis, etc.).

                    <p>This book is written by <cite>J.K. Rowling</cite>.</p>

31. What is the <bdi> element?
The `<bdi>` element represents text that is isolated from its surrounding for the purposes of bidirectional text formatting.

                    <p>Account: <bdi>user1234</bdi></p>

32. What is the <wbr> element?
The `<wbr>` element represents a word break opportunity, allowing the browser to insert a line break at that location if needed.

                    <p>This is a verylongwordthat<wbr>needsabreak.</p>

33. What is the <kbd> element?
The `<kbd>` element represents user input, typically keyboard input.

                    <p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy.</p>

34. What is the <samp> element?
The `<samp>` element represents sample output from a computer program.

                    <p>Output: <samp>Hello, World!</samp></p>
